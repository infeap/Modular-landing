# Screenshots

Placera följande skärmdumpar här för att visa dem på hemsidan:

## Nödvändiga filer:

### 1. `hem-oversikt.png`
**Var:** Hero-sektionen (överst på sidan)
**Källa:** Screenshot från Modular-appen som visar huvudöversikten med de 3 modulerna (Dokumentmallar, Mappstruktur, Pomodoro)
**Beskrivning:** Den första bilden du skickade - hem-skärmen med "Moduler" och de tre korten

### 2. `dokumentmallar.png`
**Var:** Features-sektion, första modulkortet
**Källa:** Screenshot från Dokumentmallar-modulen som visar kategorier (Ekonomi/Anbud, Projektering, Rapport, Beskrivningar)
**Beskrivning:** Den tredje bilden - dokumentmallar med kategorival och dropdown för malltyp

### 3. `mappstruktur.png`
**Var:** Features-sektion, andra modulkortet
**Källa:** Screenshot från Mappstruktur-modulen
**Beskrivning:** Vy som visar mappstruktur-funktionalitet

### 4. `pomodoro.png`
**Var:** Features-sektion, tredje modulkortet
**Källa:** Screenshot från Pomodoro-timer-modulen
**Beskrivning:** Den fjärde bilden - Pomodoro Timer med cirkel som visar 24:58 och "Arbetspass"

## Kommande moduler (visas som blurrade med "Kommer snart"):
- Kabeldimensionering
- Effektberäkningar
- Gruppförteckningar

Dessa behöver inga bilder - de visas automatiskt som "Under utveckling".

## Bildformat:
- Format: PNG eller JPG
- Upplösning: Minst 1200px bredd för bästa kvalitet
- Aspect ratio: 16:9 rekommenderas

## Så här lägger du till bilderna:
1. Ta screenshots från Modular-appen
2. Spara dem med namnen ovan i denna mapp
3. Uppdatera sidan - bilderna laddas automatiskt!
